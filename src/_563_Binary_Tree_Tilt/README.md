###### Binary Tree Tilt

二叉树的倾斜度

###### 题意
1.返回整棵树的倾斜度
2.倾斜度的定义是 左子树总和 与右子树总和 的差的绝对值。 
3.null节点为0;


###### 解题思路
1.遍历所有左/右节点的值
2.求左节点的和和右节点的和的差
3.如果结果小于0结果*-1
->错误 ：没读明白题意

###### 题意没读明白啊
实际上
倾斜度是计算 当前节点下 所有左子树和所有右子树的差。
###### 解题思路
1。遍历 
2。返回当前节点 对应的所有节点的和
3.计算倾斜度
4.倾斜度求和
